#伪元素
- 页面中的特殊位置,为其添加内容
- 内容通过CSS添加，不算网页的正式内容
- 通过它可以为网页添加统一的符号
- 也可在特殊场景下来调整页面的样式
- 以`::`开头
##`::before`
- 元素开始的位置（开始标签后）
- 例
    ```
    div::after{
        content:"after";
        color: red;
    }
    ```
##`::after`
- 元素结束的位置（结束标签之前）
##`::first-letter`首字母
##`::first-line`首行
##`::selection`选中的文字样式
